Create Pydantic v2 JSONL schema models that define the machine-readable data contracts for election data import, and write comprehensive tests proving they validate correctly.

Output: 4 Pydantic JSONL model files + 1 shared enum file + test suite in `tests/unit/test_schemas/test_jsonl_schemas.py`
