    - "2026-1-5" -> "01/05/2026" (ISO with single digits)
    - "--" -> "--" (placeholder unchanged)
    - "invalid" -> "invalid" (unparseable unchanged)
    - Target format "iso": "01/05/2026" -> "2026-01-05"
